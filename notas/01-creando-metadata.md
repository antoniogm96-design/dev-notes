# Creando el archivo metadata.json

Siguiendo las instrucciones del hackathon, el primer paso fue crear un repositorio público en GitHub.

Luego, creé el archivo `metadata.json` con mi información básica:
- owner: Mi nombre
- contact: Enlaces a mi GitHub y LinkedIn

Dejé placeholders para la URL de mi avatar y la URL de la lista de notas, que actualizaré más adelante. Este archivo es la base para que otros desarrolladores puedan encontrar mis notas.
