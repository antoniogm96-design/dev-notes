# Markdown con Frontmatter

El reto especifica que cada nota debe ser un archivo Markdown con Frontmatter.

- El **Frontmatter** (entre los `---`) son los datos para la máquina: título, fecha, resumen. Esto permite que otros programas puedan listar mis notas.
- El **Markdown** (el resto del archivo) es el contenido para que lo lean las personas.

Es una solución brillante y simple. Permite que un solo archivo de texto sirva como su propia fuente de datos y contenido legible.
